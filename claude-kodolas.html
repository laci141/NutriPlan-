Claude Code — Terminál (CLI)

Legjobb választás

A legerősebb módja a Claude-dal való fejlesztésnek. Közvetlenül a projekted mappájában dolgozik.

  • Teljes projekthozzáférés, git integráció
  • Tesztek futtatása, hibakeresés automatikusan
  • Nagy refaktorálások, több fájlos módosítások
Mikor válaszd? ▼
Ha valódi szoftverprojekten dolgozol: webapp, API, mobilalkalmazás. A CLI látja a teljes kódbázist, futtatja a parancsokat, és addig iterál a hibákon, amíg minden működik. Skillekkel és MCP szerverekkel tovább bővíthető.
🧩

VS Code / JetBrains bővítmény

IDE-ben élőknek

A Claude Code beépül a kedvenc szerkesztődbe. Látod a diffeket, egy kattintással elfogadod a módosításokat.

  • Inline diff megjelenítés a szerkesztőben
  • Kijelölt kód alapján kérdezhetsz
  • VS Code, Cursor, IntelliJ, PyCharm
Mikor válaszd? ▼
Ha már van bejáratott IDE-munkafolyamatod, és nem akarsz terminál és szerkesztő között ugrálni. A módosítások vizuálisan, diff nézetben jelennek meg — kényelmesebb átnézni, mint a nyers terminál kimenetet.
🌐

Claude Code Web

Telepítés nélkül

Böngészőből indítasz felhőbeli munkameneteket a GitHub repóidon — pont így készült ez az oldal is.

  • Fut a felhőben, a géped pihenhet
  • Több feladat párhuzamosan
  • PR-t készít, CI-t figyel helyetted
Mikor válaszd? ▼
Ha GitHubon van a projekted, és bárhonnan — akár telefonról — szeretnél feladatot kiadni. Ideális párhuzamos munkára: több munkamenet fut egyszerre, mindegyik a saját branchén, te pedig csak a kész PR-eket nézed át.
✨

Claude.ai — Artifacts

Gyors prototípus

Chat felületen kérsz egy appot, és azonnal élőben látod a chat mellett.

  • Azonnali élő előnézet
  • HTML, React, SVG egy lépésben
  • Megosztható link egy kattintással
Mikor válaszd? ▼
Ha gyorsan akarsz látni valamit: egy landing oldal ötletet, kis játékot, kalkulátort, diagramot. Nem kell hozzá semmilyen fejlesztői környezet — a böngésző és egy jó prompt elég.
🖥️

Claude Desktop alkalmazás

Mac / Windows

Asztali alkalmazás MCP-integrációkkal: fájlok, naptár, adatbázisok.

  • Helyi fájlokhoz való hozzáférés
  • MCP szerverek egyszerű beállítása
  • Claude Code is elérhető benne
Mikor válaszd? ▼
Ha a Claude-nak a helyi gépeddel kell dolgoznia: dokumentumokat rendezni, táblázatokat elemezni, vagy MCP-n keresztül más appokkal (Slack, naptár, adatbázis) együttműködni.
🤖

Claude API + Agent SDK

Fejlesztőknek

Saját alkalmazásba építenéd a Claude-ot? Az API-val saját AI-ügynököket készíthetsz.

  • Saját chatbotok, automatizálások
  • Python és TypeScript SDK
  • Tool use, MCP, streaming támogatás
Mikor válaszd? ▼
Ha terméket építesz: ügyfélszolgálati botot, dokumentum-feldolgozót, saját kódoló ügynököt. Az Agent SDK ugyanazt a motort adja a kezedbe, ami a Claude Code-ot is hajtja.

Nézd az animációt: így épül fel egy erős kérés, elemről elemre.

claude — prompt írás élőben
> ✓ Erős prompt — a Claude pontosan tudja, mit kérsz!

Profi tippek 2026-ra

Koppints egy tippre a részletes magyarázatért.

1

Készíts CLAUDE.md fájlt

▼
A projekt gyökerébe tett CLAUDE.md-ben leírhatod a szabályokat, build parancsokat, kódolási konvenciókat — a Claude minden munkamenet elején elolvassa, így nem kell újra elmagyaráznod. Generáltathatod is a /init paranccsal.
2

Adj pontos, konkrét feladatot

▼
Minél konkrétabb a kérés (fájlnév, elvárt viselkedés, példa bemenet és kimenet), annál jobb az eredmény. A homályos kérés homályos kódot szül. Képernyőképet is csatolhatsz — a Claude látja és érti.
3

Használj plan módot nagy feladatnál

▼
Összetett módosítás előtt kérj tervet — így előbb átbeszélitek a megközelítést, és csak jóváhagyás után nyúl a kódhoz. Ez sok felesleges kört spórol meg nagy refaktorálásoknál.
4

Kérj teszteket is

▼
„Írd meg és futtasd is a teszteket" — a Claude Code képes futtatni a tesztjeidet és addig javítani a kódot, amíg minden zöld nem lesz. Így a kész munka valóban működik, nem csak jól néz ki.
5

Dolgozz külön brancheken

▼
Hagyd, hogy a Claude feature branchen dolgozzon és PR-t nyisson — így mindig átnézheted a változásokat merge előtt, és bármikor visszaléphetsz, ha valami nem tetszik.
6

MCP-vel bővítsd a képességeit

▼
MCP szerverekkel a Claude eléri az adatbázisodat, a Slacket, a naptáradat, a böngészőt — így nem csak kódol, hanem a teljes munkafolyamatodban segít, a hibajegytől a deployig.

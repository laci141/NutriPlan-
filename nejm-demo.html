Installation & Setup

Termux / Android / Linux
1
Clone & Build
BASH
cd ~/printing-press-library/library/health/nejm
GOTOOLCHAIN=local go build -o nejm ./cmd/nejm-pp-cli/
2
Sync RSS Feeds
BASH
# Current issue table of contents
./nejm sync --feed etoc
# Ahead-of-print articles
./nejm sync --feed axatoc
3
Verify
BASH
./nejm doctor

db_size     602 KB
schema      v4
articles    135
status      OK ✓
4
MCP Server (optional)
BASH
# Add to Claude Desktop
claude mcp add nejm-pp-mcp -- nejm-pp-mcp

Trends & Open Access

Requires --enrich
--ENRICH Trend analysis

Analyzes publication trends over time. Requires per-article enrichment to populate specialty and type metadata.

BASH
# Sync first, then enrich + trend
./nejm sync --feed etoc
./nejm trends --enrich --human-friendly
./nejm trends --enrich --json

⚠ Without --enrich, output may be empty (no enriched records).

--ENRICH Open access filter

Show only freely available articles. Enrichment fetches access status per DOI.

BASH
# Enrich article to mark access
./nejm article 10.1056/NEJMicm2600597 --enrich

# Then filter open access
./nejm open-access --enrich --human-friendly
./nejm open-access --enrich --json

⚠ Must run article --enrich for each DOI before open-access shows results.

Reading List

Personal article queue
LOCAL Manage your reading list

Save articles to a personal reading queue by DOI. Persists in local DB.

BASH
# Add an article
./nejm reading-list add 10.1056/NEJMicm2600597

# View your list
./nejm reading-list --human-friendly

# Remove an article
./nejm reading-list remove 10.1056/NEJMicm2600597

# JSON output
./nejm reading-list --json
SQL Raw SQL queries

Run arbitrary SQL against the local SQLite database for custom analysis.

BASH
# Count total articles
./nejm sql "SELECT COUNT(*) FROM articles"

# Latest 5 titles
./nejm sql "SELECT title, published_at FROM articles ORDER BY published_at DESC LIMIT 5"

# Articles per month
./nejm sql "SELECT strftime('%Y-%m', published_at) AS month, COUNT(*) FROM articles GROUP BY month"

Workflow

Multi-step compound operations
COMPOUND Workflow command

Chain multiple operations into a single natural-language command.

BASH
./nejm workflow "sync and show current issue" --human-friendly
./nejm workflow "search cancer and summarize" --human-friendly
./nejm workflow "digest this week" --human-friendly
WHICH Natural language → command

Resolve a plain-English question into the correct CLI subcommand.

BASH
./nejm which "show me latest heart articles"
→ specialty "Cardiology" --human-friendly

./nejm which "find recent cancer research"
→ search "cancer" --human-friendly

./nejm which "what was published this week"
→ since 2026-06-18 --human-friendly

Output Modes

Flags for every use case
HUMAN--human-friendly

Colored, formatted terminal output for reading in the shell.

BASH

JSON--json

Machine-readable JSON output for scripts, piping, or agent integration.

BASH
./nejm search "cancer" --json
AGENT--agent

JSON + no color + no prompts. Designed for CI/CD pipelines and Claude MCP.

BASH
./nejm current --agent

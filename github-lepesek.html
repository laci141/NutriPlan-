1. lépés

GitHub megnyitása

Chrome → Írj a címsávba:

github.com
💡 Ha még nincs fiókod → koppints Sign up
Ha már van fiókod → koppints Sign in
📁

2. lépés

Repository megnyitása

Írd be egyenesen ezt:

💡 Még nincs repó?
Jobb felül + ikon → New repository → Adj nevet → Public → Create repository
⬆️

3. lépés

HTML fájl feltöltése

Közvetlen feltöltési link:

✅ Megnyílik a feltöltő oldal. Koppints:
"choose your files" → Fájlok / Letöltések mappa → válaszd ki az index.html-t
💾

4. lépés

Mentés véglegesítése

Miután a fájl megjelent (zöld pipa):

  1. 1 Görgess le az oldal aljára
  2. 2 Keresd a "Commit changes" szekciót
  3. 3 Ellenőrizd: "Commit directly to main branch" legyen kijelölve
  4. 4 Koppints a zöld "Commit changes" gombra
✅ A fájlod felkerült a GitHub-ra!
⚙️

5. lépés

Weboldal bekapcsolása (Pages)

⚠️ Csak EGYSZER kell! Menj ide:

  1. 1 Koppints a "None" legördülőre → válaszd: main
  2. 2 Mappa maradjon: / (root)
  3. 3 Koppints a Save gombra
  4. 4 Frissítsd az oldalt 1-2 perc múlva
🔗

6. lépés

Megkaptad az élő linket!

Az appod élő, publikus linkje:

Kezdőképernyőre rögzítés:

  1. 1 Chrome-ban nyisd meg a fenti linket
  2. 2 ⋮ → Hozzáadás a kezdőképernyőhöz
  3. 3 Hozzáadás → 🎉 Kész, appként nyílik!
🔄

Frissítés

Új verzió feltöltése

Ha Claude-dal módosítottad az appot:

  1. 1 Töltsd le az új index.html-t Claude-tól
  2. 2 Nyisd meg a fenti linket
  3. 3 Válaszd ki az új fájlt → Commit changes
ℹ️ Pages beállítást nem kell megismételni!
1-2 perc múlva automatikusan frissül az appod.

Consensus live API · 5 sources
Physicians · Science Journalists · Fact-Checkers · Patients

Score any scientific claim from 0 to 100. Aggregates evidence from all 5 sources, classifies stance (SUPPORTS / REFUTES / MIXED), and reports confidence level and evidence strength. The fastest answer to "what does the science say about X?"

Q1 · Vaccine-autism myth
./scientific-consensus consensus "vaccines cause autism" --human-friendly
Q2 · Statins cardiovascular
./scientific-consensus consensus "statins reduce cardiovascular risk" --human-friendly
Q3 · Mediterranean diet — JSON output
./scientific-consensus consensus "mediterranean diet prevents heart disease" --json

Score meaning: 0–20 = strongly refuted · 21–40 = probably false · 41–60 = mixed / contested · 61–80 = supported · 81–100 = strongly supported. Use --json for pipeline-compatible output.
When to use
A science journalist needs to fact-check whether "sugar is more addictive than cocaine." consensus "sugar addiction cocaine" returns score 28/100 (MIXED, LOW confidence) with 340 studies — the journalist learns the claim is contested but not debunked, guiding a nuanced article.

Evidence Pyramid live API · study classification
Clinicians · Systematic Reviewers · Evidence-Based Medicine · Students

Classifies all studies on a topic by design — meta-analysis, RCT, cohort, case-control, case report — and renders an evidence pyramid. Immediately shows where evidence is strong (many RCTs at the top) vs weak (mostly expert opinion at the base).

When to use
A clinical guideline committee reviewing "intermittent fasting for type 2 diabetes" needs to grade evidence strength. evidence "intermittent fasting weight loss" shows 14 meta-analyses, 38 RCTs, 127 cohort studies — the committee grades it MODERATE evidence (Grade B).

Export & Curate 5 formats · reading lists
Researchers · Reference Managers · Journal Editors · Librarians

export outputs resource lists in 5 formats — markdown, JSON, CSV, HTML, BibTeX. First argument is always the resource type (works / authors / institutions / sources / funders), not a topic string. curate generates a ranked, deduplicated reading list combining the best results from all 5 sources.

Q37 · Export works as BibTeX (Zotero / Mendeley / LaTeX)
./scientific-consensus export works --format bibtex
Q38 · Export authors as CSV
./scientific-consensus export authors --format csv
Q39 · Curated reading list — microbiome mental health
./scientific-consensus curate "microbiome mental health" --limit 20 --human-friendly

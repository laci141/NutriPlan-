Consensus

Evidence stance score
LIVE APIconsensus command

Evaluates a claim across all sources. Returns score (0–100), confidence level, stance (supports / refutes / mixed), and evidence strength.

BASH
./scientific-consensus consensus "vaccines cause autism" --human-friendly
./scientific-consensus consensus "mediterranean diet prevents heart disease" --human-friendly
./scientific-consensus consensus "statins reduce cardiovascular risk" --human-friendly
./scientific-consensus consensus "coffee protects against Parkinson" --json

Compare

Side-by-side claim analysis
LIVE APIcompare command

Runs parallel evidence analysis on two competing claims or treatments.

BASH

./scientific-consensus compare "ozempic weight loss" "lifestyle intervention weight loss" --human-friendly
./scientific-consensus compare "low-carb diet" "low-fat diet" --human-friendly

Trends & Monitoring

Growth, drift, emerging, watch
LIVE APItrends + emerging + timeline

Year-over-year growth analysis, fastest-growing areas, and publication timeline with milestone detection.

BASH
./scientific-consensus trends "GLP-1 obesity" --human-friendly
./scientific-consensus trends "artificial intelligence radiology" --human-friendly
./scientific-consensus emerging --limit 10 --human-friendly
./scientific-consensus timeline "mRNA vaccines" --human-friendly
./scientific-consensus timeline "CRISPR gene editing" --human-friendly
LIVE APIdrift + watch

Compare topic distributions across time periods. Monitor for major new publications.

BASH
# drift requires min 2-year span
./scientific-consensus drift "cancer immunotherapy" --from 2018 --to 2024 --human-friendly
./scientific-consensus drift "covid-19 long term effects" --from 2020 --to 2025 --human-friendly

# watch for new publications
./scientific-consensus watch "long covid" --human-friendly
./scientific-consensus watch "alzheimer amyloid" --human-friendly

Output Modes & Key Flags

Flags for every use case
HUMAN--human-friendly

Colored, formatted terminal output with visual elements.

BASH
./scientific-consensus consensus "aspirin" --human-friendly
JSON--json / --agent

Machine-readable JSON. --agent adds no-color + no-input for CI/scripts.

BASH
./scientific-consensus consensus "aspirin" --json
./scientific-consensus consensus "aspirin" --agent
AI--summarize

AI-enhanced summaries. Uses S2 TLDR as fallback without API keys.

BASH
./scientific-consensus search "alzheimer" --summarize --human-friendly
